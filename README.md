# 🧠 ReconFlow AI — Multi-Agent Invoice Reconciliation System

## 🚀 Overview
ReconFlow AI is an AI-powered multi-agent system that automates invoice reconciliation using structured reasoning, tool-calling, and rule-based decision intelligence. It simulates enterprise finance workflows like invoice validation, PO matching, compliance checks, and automated vendor communication.

## 🏗️ Features
- Multi-Agent Architecture (Invoice, PO, Decision, Email, Tool Planner)
- Tool-Calling Execution Engine (PO lookup + amount validation)
- Rule-based Decision System (Approve / Reject / Escalate)
- RAG-style Policy Retrieval (compliance-aware decisions)
- Explainable AI logs for audit transparency
- Streamlit dashboard for interactive workflow execution
- Analytics engine for reconciliation insights

## 📊 Workflow
Invoice → PO Lookup → Validation Tools → Decision Engine → Email Generation → Analytics

## 🧠 Key Concepts
- Agentic AI Systems
- Tool Calling / Function Execution
- Retrieval-Augmented Decision Making (RAG)
- Workflow Automation
- Explainable AI (XAI)

## 📈 Impact
- Automates invoice reconciliation pipeline
- Reduces manual finance workload
- Ensures compliance-aware decision making
- Provides full traceability of financial decisions
